# Advanced-SQL-Interview-Preparation
# SQL Interview Questions & Advanced Query Practice

This repository contains a collection of advanced SQL interview questions and solutions focused on real-world business scenarios and analytical problem-solving.

## 📌 Topics Covered

- Common Table Expressions (CTEs)
- Window Functions
- LAG() Function
- ROW_NUMBER()
- RANK()
- DENSE_RANK()
- Month-over-Month (MoM) Growth
- Year-over-Year (YoY) Growth
- Top-N Analysis
- Sales Analytics
- Employee Salary Ranking
- Product Performance Analysis

## 📂 Database Used

MySQL 8.0+

The script creates sample databases and tables with sample data for practicing SQL concepts.
